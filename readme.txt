Gross Salary Calculator Program:

This C++ program was written as part of my learning journey to understand the basics of C++ programming. The program calculates the gross salary of an individual based on their working hours and hourly wage.

Program Overview:

- Prompts the user to input their working hours
- Prompts the user to input their hourly wage in rupees
- Calculates the gross salary by multiplying the working hours and hourly wage
- Displays the gross salary in rupees

Features:

- User input for working hours and hourly wage
- Use of multiplication operator for gross salary calculation
- Clear and informative output

How to Run the Program:

1. Copy the provided C++ code into a .cpp file
2. Compile the program using a C++ compiler
3. Run the compiled program:
    - On Linux/macOS: ./gross_salary_calculator
    - On Windows: gross_salary_calculator.exe
4. Enter your working hours and hourly wage when prompted

Learning Objectives:

- Understanding basic user input and output
- Using integer variables and arithmetic operators
- Working with basic salary calculation logic
- Basic program structure and syntax
